## 기능 목록 ver 1.0

<hr>

### 예외 처리 목록
- 다리 길이
  - 양의 정수가 아닌 경우
  - 3 ~ 20 사이의 값이 아닌 경우


- 플레이어가 이동할 칸
  - U or D 가 아닌 다른 값인 경우

<hr>

### 기능 목록

- 주어진 기능
  - 0, 1 랜덤 값 생성

 
- 입력 기능 (UI 로직)
  - 다리 길이 입력받기
  - 이동할 칸 입력받기
  - 게임 재시작 여부 입력받기
  

- 출력 기능 (UI 로직)
  - 게임 시작 문구
  - 다리 길이 입력받기 문구
  - 이동할 칸 입력받기 문구
  - 게임 재시작 여부 입력받기 문구
  - 최종 게임 결과 다리 출력
  - 게임 성공 여부 출력 문구
  - 총 시도 횟수 출력 문구
  - 다리 건너기 결과 출력
    - O (성공)
    - X (실패)


- 비즈니스 로직
  - 다리 생성
  - 건널 수 있는 다리인지 확인
    - 생성된 다리와 사용자의 선택 비교
      - 게임 오버
      - 게임 진행
  - 최종 도착 지점 확인
  - 총 시도 횟수 저장
  - 게임 성공 여부 확인
